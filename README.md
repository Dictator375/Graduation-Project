## هيكل المشروع / Structure du projet

```
Graduation-project/
├── database/           ← قاعدة البيانات (SQLite)
│   ├── schema.sql      ← جداول قاعدة البيانات
│   └── seed.sql        ← البيانات الأولية
├── backend/            ← الخادم (Node.js + Express)
│   ├── server.js       ← نقطة الدخول الرئيسية
│   ├── routes/         ← مسارات API
│   ├── middleware/     ← المصادقة والصلاحيات
│   └── models/         ← قاعدة البيانات
├── desktop/            ← تطبيق سطح المكتب (Electron + React)
│   ├── main.js         ← عملية Electron الرئيسية
│   └── src/            ← واجهة React
├── mobile/             ← تطبيق الهاتف (React Native + Expo)
│   ├── App.js          ← نقطة الدخول
│   └── src/            ← الشاشات والمكونات
└── shared/             ← كود مشترك
    ├── i18n/           ← ترجمات (عربي / فرنسي)
    └── constants/      ← ثوابت مشتركة
```

## متطلبات التثبيت / Prérequis

1. **Node.js** — https://nodejs.org  (version 18 ou supérieure)
2. **Git** — https://git-scm.com
3. **VS Code** — https://code.visualstudio.com
4. **Expo Go** sur votre téléphone — depuis le Play Store ou App Store

---

## تشغيل المشروع / Démarrage

### 1. الخادم / Backend

```bash
cd backend
npm install
npm run dev
```

✅ الخادم يعمل على: http://localhost:3001

### 2. تطبيق سطح المكتب / Desktop

```bash
cd desktop
npm install
npm run dev
```

✅ نافذة Electron ستفتح تلقائياً

### 3. تطبيق الهاتف / Mobile

```bash
cd mobile
npm install
npx expo start
```

ثم امسح QR Code بتطبيق **Expo Go** على هاتفك.

> ⚠️ إذا كنت تستخدم هاتفاً حقيقياً (وليس محاكياً)، افتح
> `mobile/src/utils/api.js` وغيّر `10.0.2.2` إلى IP جهازك
> مثال: `http://192.168.1.100:3001/api`
